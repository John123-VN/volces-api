# volces-api

An unofficial, asynchronous Python API wrapper for Jimeng (火山引擎 - Volcengine AI image and video generation platform). Built on top of `httpx` for fast, non-blocking requests.

## Installation

```bash
pip install volces-api
```

## Quick Start
Go to consoles.volcengine.com/auth/login
Log in or sign up in Volcengine.
From the main dashboard or side menu, navigate to Large Models > Volcano Ark.
Click on API Key Management (API Key 管理) in the left navigation sidebar.
Click the Create API Key (创建 API Key) button.
Provide an API key name and description, then confirm.
Copy and save the secret key immediately, as it will only be fully visible once.
